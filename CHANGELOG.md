# Changelog

All notable changes to this project are documented in this file.

## [Unreleased]

### Added
- New checklist content and guidance, including LLM-related harms, positive examples, and human review items.

### Changed
- Modernized packaging setup to use PEP 621 metadata.

### Fixed
- HTML appending compatibility for `beautifulsoup4>=4.12.1`.

## [0.3.0] - 2020-12-11

### Added
- New checklist item for downstream bias mitigation.

### Changed
- Brazilian Portuguese checklist text updates.
- Minor formatting updates in checklist content.

## [0.2.2] - 2020-07-02

### Added
- Additional checklist examples and reference links.
- Expanded Brazilian Portuguese content and translation updates.
- Conda usage information.

### Fixed
- Main-call behavior for script execution.

## [0.2.1] - 2020-02-07

### Added
- Jupyter multicell output format.
- `LICENSE.txt` inclusion in source distributions.

## [0.2.0] - 2020-01-06

### Changed
- Jupyter and HTML render method updates.
- Clipboard functionality (and `xerox` dependency) removed.

## [0.1.4] - 2019-10-21

### Added
- Documentation improvements, including badge usage instructions and asciinema references.

### Changed
- Dependency floor updates (`jinja2>=2.10.1`).

### Fixed
- YAML loading hardened by switching to `SafeLoader`.
